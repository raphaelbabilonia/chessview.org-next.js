# Coverage map boundary data

`admin1-boundaries.json` is generated by `npm run maps:build-admin1` from Natural Earth Admin-1 states/provinces lines at the pinned upstream commit recorded in the file.

Natural Earth data is public domain. The generated asset retains only country names and simplified, quantized boundary coordinates. It is loaded lazily after users zoom into the coverage map.

Source: https://www.naturalearthdata.com/downloads/10m-cultural-vectors/10m-admin-1-states-provinces/
